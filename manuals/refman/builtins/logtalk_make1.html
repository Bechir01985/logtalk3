logtalk_make/1

Description

logtalk_make(Target)

Allows reloading all Logtalk source files that have been modified since the time they are last loaded when called with the argument all and deleting all intermediate files generated by the compilation of Logtalk source files when called with the argument clean.

There are some caveats when using this built-in predicate, however, using the all target. First, only Logtalk source files are reloaded. Second, source files are reloaded using the same explicit compiler options used the last time they are loaded but no check is made if the implicit compiler options have changed between loads.

Template and modes

logtalk_make(+atom)

Errors

(none)

Examples

| ?- logtalk_make(clean).
