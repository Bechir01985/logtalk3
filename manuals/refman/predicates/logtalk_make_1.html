logtalk_make/1

Description

logtalk_make(Target)

Allows reloading all Logtalk source files that have been modified since last loaded when called with the target all and deleting all intermediate files generated by the compilation of Logtalk source files when called with the target clean.

When using the all target, only source files loaded using the logtalk_load/1-2 predicates are reloaded. Non-modified files will also be reloaded when there is a change to the compilation mode (i.e. when the files were loaded without explicit debug/1 or optimize/1 options and the default values of these options changed after loading; no check is made, however, for other implicit compiler options that may have changed since loading).

Template and modes

logtalk_make(+atom)

Errors

(none)

Examples

| ?- logtalk_make(clean).
