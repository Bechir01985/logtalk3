This folder is used as a scratch folder for the temporary files
generated by the Logtalk compiler when compiling source files.

You may set the value of the `scratch_directory` to this folder
if you want to collect all temporary files in the same location.

If you delete this folder by accident, it will be recreated the
next time you run Logtalk.
