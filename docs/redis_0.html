Remarks

Command representation
Use the Redis command name as the functor of a compound term where the arguments are the command arguments.
Valid arguments
Atoms, integers, and floats. Always use atoms instead of double-quoted "strings". This helps portability by not depending on the value of the double_quotes flag.
