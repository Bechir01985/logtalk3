approximate_equality/4

Compares two floats for approximate equality using the provided relative and absolute tolerances using the de facto standard formula abs(Float1 - Float2) =< max(RelativeTolerance * max(abs(Float1), abs(Float2)), AbsoluteTolerance). No type-checking.

Compares two floats (or lists of floats) for approximate equality using 100*epsilon for the absolute error and, if that fails, 99.999% accuracy for the relative error. Note that these precision values may not be adequate for all cases. No type-checking.
