This directory contains files that provide support for using version 3.0 
or later of the jQuery.Syntax package by Samuel Williams with Logtalk 
source files. A detailed description on the jQuery.Syntax package is 
available from:

	http://www.oriontransfer.co.nz/software/jquery-syntax/index.en

These support files are dual-licensed under the GNU General Public License 3 
and the jQuery.Syntax license.

In order to check if your jQuery.Syntax distribution already includes 
support for Logtalk, look for a file named "jquery.syntax.brush.logtalk.js"
in the "source" and "public" folders. If support for Logtalk is not included
or if it is outdated, see the follwing URL for instructions on how to add
the Logtalk brush file:

	http://www.oriontransfer.co.nz/software/jquery-syntax/how-to-install/advanced-setup/index.en

The file "source.html" is a test file based on the "../tests/source.lgt"
file.
